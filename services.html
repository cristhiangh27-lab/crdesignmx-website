Process / Method

A disciplined path from first conversation to site strategy.

  1. 01

    Brief

    We define scope, location, project stage, constraints and the decisions that must be resolved first.

  2. 02

    Concept

    We establish spatial direction, priorities, atmosphere and the main design criteria.

  3. 03

    BIM model

    We organize the project into coordinated information that supports drawings, quantities and reviews.

  4. 04

    Visualization

    We test how the project is read through material, light, scale and presentation imagery.

  5. 05

    Construction strategy

    We translate design decisions into sequencing, site priorities and practical next steps.
